# Doctrine ODM

**Language / Ecosystem:** PHP
**Repository:** [doctrine/mongodb-odm](https://github.com/doctrine/mongodb-odm)

Doctrine MongoDB ODM is a PHP library that provides transparent persistence for PHP objects to MongoDB. It is built on top of the official MongoDB PHP library and integrates with the Doctrine project's common abstractions for annotations/attributes, events, and the Unit of Work pattern.
