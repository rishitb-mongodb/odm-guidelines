# Django (Djongo / MongoEngine)

**Language / Ecosystem:** Python / Django
**Repository:** [MongoEngine/mongoengine](https://github.com/MongoEngine/mongoengine)

The Python ecosystem has multiple MongoDB ODM options. MongoEngine is a document-object mapper built on top of PyMongo, designed to integrate with the Python ecosystem. Djongo provides a Django ORM backend that allows standard Django models to be used with MongoDB. This page covers MongoDB's guidelines triage for the Python ODM surface.

## Guidelines Triage

| Feature | Triage Status |
|---------|---------------|
| [Query Builder](../guidelines/query-builder/query-builder.md) | ![Done](https://img.shields.io/badge/Done-brightgreen?style=flat) — `QuerySet` API with `filter()`/`exclude()`/`order_by()` chaining; Aggregation Pipeline via `.aggregate()`; `__raw__` keyword for raw MQL; when a queryset has active filters, MongoEngine prepends stages before user pipeline |
| [Bulk Operations](../guidelines/bulk-operations/bulk-operations.md) | ![Backlog](https://img.shields.io/badge/Backlog-lightgrey?style=flat) — `Document.objects.insert()` for batch inserts; upsert via `QuerySet.modify()`; no `bulk_write` wrapper — ordered/unordered `BulkWrite` requires raw PyMongo via `_get_collection()` |
| [Relationship Mapping](../guidelines/relationship-mapping/relationship-mapping.md) | ![Done](https://img.shields.io/badge/Done-brightgreen?style=flat) — `EmbeddedDocumentField` and `ReferenceField` with lazy/eager dereferencing; `LazyReferenceField`; `no_dereference()` context manager; `NULLIFY`/`CASCADE`/`DENY`/`PULL` delete rules; `ReferenceField` inside `EmbeddedDocument` has limitations |
| [Index Management](../guidelines/index-management/index-management.md) | ![Done](https://img.shields.io/badge/Done-brightgreen?style=flat) — Meta `indexes` declaration; single-field, compound, text (`$` prefix), hashed (`#` prefix), TTL (`expireAfterSeconds`), sparse, unique, geospatial; auto-creation configurable via `auto_create_index` and `auto_create_index_on_save` |
| [Transactions](../guidelines/transactions/transactions.md) | ![Done](https://img.shields.io/badge/Done-brightgreen?style=flat) — `run_in_transaction()` context manager/decorator (v0.29+); requires replica set; auto-retry only for `UnknownTransactionCommitResult`, not `TransientTransactionError` |
| [BSON Data Type Support](../guidelines/bson-data-types/bson-data-types.md) | ![Done](https://img.shields.io/badge/Done-brightgreen?style=flat) — `ObjectIdField`, `DateTimeField`, `Decimal128Field`, `UUIDField` (binary or string), `IntField` (32-bit), `LongField` (64-bit), `FloatField`; `ComplexDateTimeField` for microsecond precision; `null=True`/`required=False` may omit field rather than store null |
| [Array Fields](../guidelines/array-fields/array-fields.md) | ![Done](https://img.shields.io/badge/Done-brightgreen?style=flat) — `ListField`/`SortedListField`; `$in`, `$all`, `$elemMatch`, `$size` via double-underscore syntax; `$push`, `$pull`, `$addToSet`, `$pop` via `modify()`; `arrayFilters` via `__raw__`; `SortedListField` has race conditions under concurrent writes |
| [Embedded Fields](../guidelines/embedded-fields/embedded-fields.md) | ![Done](https://img.shields.io/badge/Done-brightgreen?style=flat) — `EmbeddedDocumentField`; double-underscore dot notation (`embedded__field`); `$set` partial updates via `update(set__embedded__field=value)`; complex nested updates require `__raw__` |
| [Polymorphic Array/Embedded Fields](../guidelines/polymorphic-fields/polymorphic-fields.md) | ![Done](https://img.shields.io/badge/Done-brightgreen?style=flat) — `_cls` discriminator with `allow_inheritance=True` (not default); `_cls` field auto-indexed; type-based filtering; `allow_inheritance` must be explicitly enabled per document class |
| [Security Features (CSFLE & QE)](../guidelines/security-features/security-features.md) | ![Won't Do](https://img.shields.io/badge/Won't_Do-red?style=flat) — No MongoEngine-level CSFLE or QE integration; must be configured at PyMongo driver level using `autoEncryption` |
| [Joins ($lookup)](../guidelines/joins/joins.md) | ![Done](https://img.shields.io/badge/Done-brightgreen?style=flat) — `$lookup` via `.aggregate()` pipeline; sub-pipeline option supported; results are raw dicts, not MongoEngine Document instances |
| [Atlas Search](../guidelines/atlas-search/atlas-search.md) | ![Done](https://img.shields.io/badge/Done-brightgreen?style=flat) — `$search`/`$searchMeta` stages via `.aggregate()` pipeline; no wrapper API — raw pipeline dict syntax required |
| [Vector Search](../guidelines/vector-search/vector-search.md) | ![Done](https://img.shields.io/badge/Done-brightgreen?style=flat) — `$vectorSearch` stage via `.aggregate()` pipeline; ANN and ENN supported; no wrapper API — raw pipeline dict syntax required |
| [Logging](../guidelines/logging/logging.md) | ![Done](https://img.shields.io/badge/Done-brightgreen?style=flat) — PyMongo `monitoring.CommandListener` for command logging (duration in microseconds); Python `logging` module integration; no built-in slow query detection; command logs truncated at 1000 bytes by default |
| [Escape Hatch](../guidelines/escape-hatch/escape-hatch.md) | ![Done](https://img.shields.io/badge/Done-brightgreen?style=flat) — `Document._get_collection()` (protected method) for raw PyMongo `Collection`; `__raw__` for raw MQL operators; full aggregation pipeline access |
| [Performance Benchmarks](../guidelines/performance-benchmarks/performance-benchmarks.md) | ![Backlog](https://img.shields.io/badge/Backlog-lightgrey?style=flat) |
