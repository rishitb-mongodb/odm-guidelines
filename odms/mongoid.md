# Mongoid

**Language / Ecosystem:** Ruby / Ruby on Rails
**Repository:** [mongodb/mongoid](https://github.com/mongodb/mongoid)

Mongoid is a MongoDB-maintained ODM for Ruby. It provides an ActiveModel/ActiveRecord-inspired API for defining document models, querying MongoDB, and managing associations. It integrates with Ruby on Rails and Sinatra and is built on top of the official MongoDB Ruby driver.
