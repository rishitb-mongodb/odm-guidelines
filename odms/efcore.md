# EF Core (MongoDB Entity Framework Core Provider)

**Language / Ecosystem:** C# / .NET
**Repository:** [mongodb/mongo-efcore-provider](https://github.com/mongodb/mongo-efcore-provider)

The MongoDB EF Core Provider is a MongoDB-maintained database provider for Microsoft's Entity Framework Core. It allows .NET developers to use EF Core's DbContext, LINQ queries, and change tracking with MongoDB as the backing store, sitting on top of the official MongoDB C# driver.
